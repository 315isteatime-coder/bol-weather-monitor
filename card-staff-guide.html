💡 登入後你淨係見到自己崗位嘅蓋章掣，蓋到你嘅上限（門口 1、內場 2、工作坊 1）會自動鎖。同一張卡遊戲最多 4 章（門口 1 + 內場A 2 + 內場B 2 = 5 個機會，封頂 4，第 5 次顯示「此分類已滿」）。核消（換獎）淨係 center@ 大會中心帳號做。

① 登入（開工時做一次）

  1. 手機瀏覽器開 ieclong.com/card-staff
  2. 輸入你崗位嘅帳號（上表）+ 密碼 → 撳「登入」
  3. 登入一次，同一部手機之後會記住，唔使再登入
  4. 登入後頂部顯示「👤 你帳號 ｜ 你可蓋：…」，核對係咪你崗位先開始

② 幫參加者蓋章 🏅

  1. 叫參加者開「電子集點卡」（活動頁個黃色掣，或 ieclong.com/card），報佢個卡號（樣 IEC-XXXXX）
  2. 喺最上面格輸入卡號 → 撳「查卡」
  3. 見到佢張卡 → 撳你崗位嘅蓋章掣（你淨係見到自己負責嗰類）
  4. 每個掣下面寫「卡 X/上限　·　你 Y/你上限」：到你上限或卡封頂就自動變灰鎖住
  5. 參加者部手機會喺 25 秒內自動更新，唔使佢 refresh
💡 center@ 大會中心帳號見到工作坊 / 打卡 / 消費 / 商戶（唔包遊戲，工作坊可蓋 2）；其餘崗位淨係出自己嗰類。撳完想查下一位，撳「查另一張卡」。

③ 核消換獎（center@ 大會中心 · 服務處 A29–A30）

🏛 核消掣淨係 center@bolmacau.com 登入先會出；其他崗位帳號睇得卡但唔出核消掣。
  1. 用 center@bolmacau.com 登入 → 輸入參加者卡號 → 查卡
  2. 夠章嘅級數會出「換領 →」掣（未夠章嘅顯示「仲爭 X 章」）
  3. 每張卡淨係換一次：喺佢夠到嘅級數中揀一份畀佢（例如佢想要工作坊體驗就揀嗰級）
  4. 確認畀咗禮物之後，撳「換領」→ 彈確認框 → OK → 變「已換領 ✔︎」
  5. 換完即鎖卡，全部級數唔再出掣；參加者部機會自動跳去問卷頁
⚠️ 如果參加者同時用緊實體紙卡：核消電子卡之後，請喺實體卡都劃走/蓋「已換」，免佢兩邊各領一次。

❓ 常見情況

搵唔到卡？
核對卡號（IEC- 開頭 + 5 個英文/數字）。或叫參加者喺電子卡頁面重新整理一次。
參加者冇手機 / 唔想用電子卡？
照用實體紙卡蓋印，唔影響。電子卡係額外方便。
蓋錯咗想取消？
v1 暫時喺工具改唔到（只可加章），請搵大會喺後台處理。
參加者換咗手機 / 清咗瀏覽器？
電子卡會消失，以實體卡為憑。叫佢重新開一張電子卡（會係新卡號、由 0 開始）。
驚重複蓋 / 重複核消？
系統有防護：到上限自動鎖、已核消唔再出掣。放心。
🔒 唔好同參加者講登入密碼。蓋章工具只係工作人員用。
